# Миграция мастер: Racket → Common Lisp

## Контекст

Проект `development_system` состоит из:
- `мастер/` — Telegram-бот + оркестратор (сейчас на Racket)
- `подмастерье/` — executor loop (уже на CL, не трогать)
- `мастер/потоки/*.lisp` — файлы потоков (уже на CL, не трогать)

Цель: заменить мастер на Racket (`.rkt` файлы) на CL-реализацию в `мастер/` .

Новые файлы создавать в `мастер/cl/`:
```
мастер/cl/
  мастер.asd
  packages.lisp
  config.lisp
  telegram.lisp
  llm.lisp
  dusha.lisp
  история.lisp
  потоки.lisp
  комбо.lisp
  команды.lisp
  main.lisp
```

Зависимости Quicklisp: `dexador`, `cl-json`, `uiop` (встроен).

Стиль: пакет `:мастер`, функции ≤ 15 строк, рекурсия предпочтительнее loop.

Токен бота: env `MASTER_BOT_TOKEN`. Admin chat id: env `ADMIN_CHAT_ID`.
OpenRouter key: env `OPENROUTER_API_KEY`. Модель LLM: `openai/gpt-4.1`.

Пути: корень проекта — директория выше `мастер/cl/`, т.е. `мастер/`.
Потоки живут в `мастер/потоки/*.lisp`.
История диалогов: `мастер/история/<chat-id>.json`.
Душа: `мастер/душа.scm` — S-expression с полями `:имя :описание :стиль :инструкции`.
